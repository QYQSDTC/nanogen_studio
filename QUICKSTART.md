# 快速启动指南

## 第一步：安装依赖

打开终端，进入项目目录，运行：

```bash
npm install
```

## 第二步：获取 API Key

1. 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
2. 登录你的 Google 账号
3. 点击 "Create API Key" 创建一个新的 API Key
4. 复制生成的 API Key

## 第三步：启动应用

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:3008`

## 第四步：配置和使用

1. **输入 API Key**
   - 在左侧边栏的 "API Key" 输入框中粘贴你的 API Key
   - Endpoint 可以留空（自动使用 Google 官方 API）

2. **选择参数**
   - 纵横比：选择图片的宽高比（如 1:1、16:9 等）
   - 分辨率：选择 1K、2K 或 4K

3. **输入提示词**
   - 在主区域的文本框中描述你想要生成的图片
   - 例如："一只可爱的橘猫在阳光下打瞌睡"

4. **（可选）上传参考图片**
   - 点击 "上传参考图片" 按钮
   - 选择一张参考图片辅助生成

5. **生成图片**
   - 点击 "生成图片" 按钮
   - 等待几秒钟，生成的图片会显示在下方

6. **下载图片**
   - 鼠标悬停在生成的图片上
   - 点击 "下载图片" 按钮保存到本地

## 常见问题

### Q: API Key 在哪里存储？
A: API Key 仅存储在浏览器的内存中，不会上传到任何服务器，刷新页面后需要重新输入。

### Q: 生成失败怎么办？
A: 请检查：
- API Key 是否正确
- 网络连接是否正常
- 提示词是否符合 Google 的内容政策

### Q: 可以生成多张图片吗？
A: 可以！每次生成的图片都会添加到下方的网格中，你可以连续生成多张。

### Q: 支持中文提示词吗？
A: 是的，Gemini 模型支持中文和英文提示词。

### Q: 使用的是什么模型？
A: 本应用使用 `gemini-3-pro-image-preview` 模型，这是 Google 的图像生成模型，支持 `aspect_ratio` (纵横比) 和 `image_size` (分辨率) 参数。

### Q: 纵横比和分辨率参数如何工作？
A: 应用会将你选择的纵横比 (如 1:1, 16:9) 和分辨率 (1K, 2K, 4K) 作为 API 参数传递给模型，模型会严格按照这些参数生成图片。
